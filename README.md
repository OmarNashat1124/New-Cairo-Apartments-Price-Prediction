# New Cairo Apartments Price Prediction

A machine learning project that predicts apartment prices in **New Cairo** using real estate listing data. The model is trained using **CatBoost** and deployed through a simple **Streamlit** web application.

## Overview
- Predicts apartment prices based on property features  
- Uses a full preprocessing pipeline for cleaning and encoding data  
- CatBoost regression model for high-quality predictions  
- Streamlit app for easy user interaction  

## Features
- Data cleaning and feature engineering  
- Handling categorical and numerical inputs  
- Outlier removal  
- Interactive price prediction dashboard
